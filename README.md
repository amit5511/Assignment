# Required env variables<br/>

2 -> PORT<br/>
3 -> DB_URI<br/>

# Project setup<br/>
1 -> Clone the repo<br/>
2 -> Run npm install (where package.json is located)<br/>
3 -> Configure your enviornment variable<br/>
4 -> Run npm run start  to start your server<br/>
